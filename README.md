# Iodine Content Lookup

A simple Flask app for searching iodine content in common foods, with category browsing and a color-coded iodine level indicator to help users follow low-iodine diets.

## Data Attribution

Data source: U.S. Department of Agriculture, Agricultural Research Service (USDA-ARS); *USDA, FDA, and ODS-NIH Database for the Iodine Content of Common Foods*, Release 4.0 (2024). Downloaded from the USDA ARS Iodine page.

Source page: [USDA ARS Iodine](https://www.ars.usda.gov/northeast-area/beltsville-md-bhnrc/beltsville-human-nutrition-research-center/methods-and-application-of-food-composition-laboratory/mafcl-site-pages/iodine/)
